# Crypto Currency Converter

## Description
A web application that allows users to log in and convert cryptocurrencies into different currencies using live exchange rates from the CoinGecko API.

## Features

- User Login Authentication
- Authentication using users.json
- Fetch API
- Live Cryptocurrency Prices
- Cryptocurrency Conversion
- Save Favorite Currency Pairs
- Delete Favorite Pairs
- Local Storage
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- CoinGecko API
- LocalStorage
- Git & GitHub
